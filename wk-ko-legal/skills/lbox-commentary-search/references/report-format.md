# lbox-commentary-search 보고서 양식 reference

보고서 작성 직전에 이 파일을 읽고 그대로 따른다.

#### markdown 템플릿

```markdown
# 질의: <사용자의 원래 질문>

## 검색 조건

- 검색어: <키워드>
- 법령 한정: <예: 민법 채권각칙 / 미지정>
- 문서유형: <주석서 / 실무서 / 전체>
- 검색일자: <YYYY-MM-DD>
- 검색결과: 카드 N건 · 본문 visit M건 · 직접 관련 K건

## 정리된 답변

<주석서·실무서 본문을 종합해 사용자 질의에 답하는 자연스러운 문단.
각 진술 끝에 [김용덕 편, 민법 채권각칙 제6권, §섹션] 형식으로 출처 인라인 인용.
본문을 통째로 옮기지 말고 답변에 필요한 부분만 요약 + 핵심은 짧은 직접 인용("...").>

## 참고 주석서

### [도서명] 섹션 제목

- **위치(breadcrumb)**: <예: 제3편 채권 / 제5장 불법행위 / [불법행위의 효과]>
- **편집대표/저자**: ...
- **출판사 / 출판일 / 판본**: ... / ... / ...
- **관련 내용**: <질의와 관련된 부분 요약 + 필요 시 짧은 직접 인용("...")>
- **인용된 판례·법령**: <대법원 ○○○○다○○○○○, 민법 §xxx 등>
- **viewer 링크**: https://lbox.kr/book/<bookId>?tocId=<tocId>&nodeId=<nodeId>&volumeHistoryId=<volumeHistoryId>

### [도서명] 섹션 제목

...

## 주석서가 인용한 핵심 판례·법령 (있으면)

| 출처 | 인용 내용 |
|---|---|
| 대법원 ... | ... |
| 민법 §... | ... |

## 검색결과에 잡혔으나 본 질의와 직접 관련성이 낮은 섹션

- [도서명] 섹션 제목 (사유: ...)
- ...
```

#### 안내 형식

저장 후 결과 파일 경로를 안내한다.

- Cowork/데스크톱: `computer://` 링크. 예) `[검색 결과 보기](computer:///Users/.../lbox-commentary-양도금지특약-20260629.md)` (실제 작업 폴더 절대 경로로 치환, 하드코딩 금지).
- CLI 등: 절대 경로 텍스트.

#### 출처 표기 원칙

- 직접 인용·요약 모두 **도서명·저자(편집대표)·판본**까지 표기한다. 예) `[김용덕 편, 민법 채권각칙 제6권, 한국사법행정학회 제5판, 2021]`.
- 답변의 모든 진술은 visit한 본문에서 추적 가능해야 한다. 확인되지 않은 내용은 넣지 않거나 "본문에서 확인되지 않음"으로 표시한다.
