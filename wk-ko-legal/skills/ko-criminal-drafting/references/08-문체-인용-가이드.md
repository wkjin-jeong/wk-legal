# 형사 서면 문체·인용 가이드

기본 원칙은 민사 서면과 같다 — 격식체("–습니다/–합니다"), 사실은 단정형, 한자 병기 금지, 항·호·목 "1. → 가. → (1) → (가)", 날짜 "2026. 6. 10." 형식, 감정적 표현·추측 표현·이모지 금지. 여기서는 형사 서면 특유의 규칙만 정리한다.

## 1. 호칭 — 절차 단계별로 엄격히 구분

| 단계·모드 | 호칭 |
|---|---|
| 수사단계 (의견서·영장·적부심) | 피의자 |
| 공판·상소 단계 | 피고인 |
| 고소장·고발장 | 고소인(고발인) / 피고소인(피고발인) |
| 상대 당사자 기관 | 검사 (수사기관 일반은 "수사기관") |
| 피해자 | 피해자 (변호 서면에서 실명 대신 "피해자 ○○○" 첫 정의 후 "피해자") |

- 민사 용어("원고/피고")는 절대 혼입하지 않는다. "피고"와 "피고인"의 혼동이 가장 잦은 오류다.
- 공소사실은 첫 등장 시 "이 사건 공소사실"로 정의하고 이후 통일. 여러 공소사실이면 "제1 공소사실(사기의 점)", "사기의 점"으로 특정.

## 2. 법령·판례 인용

### 법령 — 행위시법 표기
- 현행본과 행위시본이 같으면 통상 표기: 형법 제347조 제1항.
- 행위 후 개정이 있었으면 **구법 표기**: 구 ○○법(2026. 1. 1. 법률 제○○호로 개정되기 전의 것) 제○조. 괄호 정보는 ko-law-api get-asof의 stderr 헤더 출력을 그대로 쓴다.
- 형법 제1조 제2항(경한 신법 우선)이 적용되는 사안이면 신·구법을 대비해 인용하고 적용 법조를 명시한다.

### 판례
민사 가이드와 동일 — "대법원 2024. 5. 8. 선고 2023도12345 판결" 형식, 전원합의체 명시, 결정은 "자 ... 결정". 사건부호(도·노·고합·고단·초적 등)를 임의로 만들지 않는다 — lbox에서 확인한 사건번호 그대로. 대표 판례의 법리 설시는 따옴표로 원문 인용(길이 제한 없음), 후속 판례는 사건번호 나열("... 등 참조").

## 3. 증거 인용 표기 (이 skill의 고정 규칙)

### 증거기록
```
(증거순번 3, 증거기록 152면)
(증거순번 7, 증거기록 2권 415면)      ← 증거기록이 1권을 넘는 경우
(증거순번 3 피의자신문조서, 증거기록 152면)   ← 증거명 병기(권장)
```
- **증거순번 + 증거기록 면수**가 기본형. 증거기록이 1권을 넘는 경우 반드시 **권수를 면수 앞에** 표기한다.
- 증거명(조서·진술서 등)을 병기하면 재판부의 확인이 빨라지므로 권장한다.
- 면수가 식별되지 않으면 추측하지 말고 "(증거순번 3, 증거기록 면수 확인 필요)"로 표시한다.

### 공판기록·판결문 등
```
(증인 ○○○에 대한 증인신문조서, 공판기록 88면)
(원심판결 5면)
(공소장 2면)
```

### 첨부 자료 (의견서·고소장 등)
```
(첨부 1 합의서) / (증 제1호 ○○계약서 — 고소장)
```
- 본문 인용과 첨부(증거) 목록의 번호 일치를 마지막에 대조한다.

## 4. 형사 서면 특유의 어조

- **공소사실 인용은 그대로, 다툼은 그 아래에서**: 공소사실을 요약·재진술하며 표현을 바꾸지 않는다.
- **사실 진술과 가정적 주장의 구분**: 예비적 정상 변론은 "가사 이 사건 공소사실이 유죄로 인정된다 하더라도"의 표지로 시작해, 부인 취지가 훼손되지 않게 한다.
- **단정의 방향**: 변호 서면에서 검사 증거의 흠은 단정적으로("...진술은 ...와 모순되어 믿기 어렵습니다"), 피고인에게 유리한 사실 중 증거가 약한 부분은 신중하게 쓴다.
- **반성·선처 문구의 절제**: 정상 변론에서도 "깊이 뉘우치고 있습니다"의 반복보다 구체적 사실(피해 회복 금액, 치료 이수, 재발 방지 조치)이 설득력 있다.
- **고소장의 어조**: 감정적 비난 대신 구성요건 중심의 건조한 기술. 처벌의사는 고소취지에서 분명하게.

## 5. 마지막 검수 — 5단계

1. 호칭이 절차 단계에 맞게 통일되었는지(피의자/피고인, 민사 용어 혼입 0건).
2. 증거 인용 표기 규칙(증거순번·권·면) 준수와 목록 일치.
3. 인용 법령의 시점(행위시법)과 구법 표기의 정확성.
4. 공소사실 인용의 원문 보존.
5. 가정적 표지("가사 ... 하더라도")의 적절한 사용.
