# 형사 서면 형식요건 체크리스트

신규 작성 전(게이트)과 초안 완성 후(자체 점검)에 두 번 사용한다. 조문은 모두 국가법령정보 API 실측 기준이며, 의문이 생기면 ko-law-api로 재확인한다.

## A. 작성 전 게이트 (하나라도 걸리면 사용자에게 먼저 알린다)

### A-1. 모드·입장
- [ ] 의뢰인이 피의자·피고인 측인가, 피해자(고소인) 측인가.
- [ ] 변호 서면이면 인부(자백/부인/일부 인부)가 확정되었는가 — 미확정이면 작성 전 확인.

### A-2. 기간

| 서면 | 기간 | 근거 |
|---|---|---|
| 항소이유서 | 소송기록접수통지 후 20일 | 형사소송법 제361조의3 |
| 상고이유서 | 소송기록접수통지 후 20일 | 형사소송법 제379조 |
| 항소 답변서 | 항소이유서 부본 송달 후 10일 | 제361조의3 제3항 |
| 친고죄 고소 | 범인을 알게 된 날부터 6월 | 제230조 |

- [ ] 상소 사건이면 소송기록접수통지일이 자료로 확인되는가. 임박 시 즉시 고지.

### A-3. 상고이유 제한 (형사소송법 제383조)
- [ ] 10년 미만의 형 선고 사건에서 사실오인·양형부당 단독 구성이 아닌가.

### A-4. 소추조건·시효
- [ ] 친고죄·반의사불벌죄 해당 여부와 고소·처벌불원 상태.
- [ ] 공소시효 완성·임박 여부(행위시법 기준 법정형으로 계산).

### A-5. 행위시법
- [ ] 범행 일시가 식별되었는가.
- [ ] 적용법조를 행위시 기준으로 확인했는가(ko-law-api get-asof). 법정형 변경이 있으면 형법 제1조 제2항(경한 신법) 검토.

## B. 서면별 필수 기재사항

### B-1. 공통
- [ ] 사건 표시(수사기관·법원 사건번호), 죄명, 피의자/피고인 표시
- [ ] 작성 명의(변호인/대리인 변호사)와 제출처(수사기관/법원/재판부)
- [ ] 작성일

### B-2. 변호인의견서 (수사)
- [ ] 구하는 처분(불송치/불기소/기소유예)이 사건 단계에 맞게 명시
- [ ] 호칭 "피의자"

### B-3. 구속 관련 서면
- [ ] 절차 단계와 서면 종류의 일치(영장심사 의견서/적부심 청구서/보석청구서)
- [ ] 적부심·보석은 청구권자 확인, 청구취지의 결정 형식
- [ ] 구속사유(보석은 제95조 제외사유) 각각에 소명 자료 대응

### B-4. 변론요지서
- [ ] 공소사실에 대한 인부 명시
- [ ] 부인+정상 변론 병행 시 가정적 표지("가사 ... 하더라도")
- [ ] 호칭 "피고인"

### B-5. 상소이유서
- [ ] 상소이유의 분류(사실오인/법리오해/양형부당/절차 위반) 항 구분
- [ ] 원심 판결문 인용에 면수 표시
- [ ] 상고이유서는 기록과 원심 증거조사에 나타난 사실만 인용(제379조 제2항)
- [ ] 사건부호(노/도)와 법원 표시의 정합

### B-6. 고소장·고발장
- [ ] 고소권자 해당 여부(제223조), 고발은 제234조
- [ ] 죄명·적용법조(ko-law-api 확인, 행위시법 기준)
- [ ] 범죄사실의 특정(일시·장소·방법·결과)과 구성요건 요소 대응
- [ ] 처벌의사 명시
- [ ] 증거 자료 목록("증 제○호")과 본문 인용 일치

## C. 완성 후 공통 점검
- [ ] 호칭 통일 — 수사 "피의자" / 공판·상소 "피고인" / 고소장 "고소인·피고소인". 민사 용어("원고·피고") 혼입 0건.
- [ ] 증거 인용 표기 — "(증거순번 ○, 증거기록 ○면)", 1권 초과 시 "(증거순번 ○, 증거기록 ○권 ○면)". 면수 불명은 "확인 필요" 표시.
- [ ] 판례 인용이 모두 lbox에서 확인된 것인가(추정 인용 0건).
- [ ] 양형기준 수치가 로컬 파일 또는 양형위원회 웹사이트에서 확인된 것인가(미확인 수치 0건).
- [ ] 합의서·공탁서·반성문 등 정상 자료가 실제 확보된 것만 인용되었는가.
- [ ] 공소사실 인용이 공소장 기재 그대로인가(무단 요약·변형 없음).
- [ ] 증거능력 주장과 증거목록상 동의 여부의 정합.
