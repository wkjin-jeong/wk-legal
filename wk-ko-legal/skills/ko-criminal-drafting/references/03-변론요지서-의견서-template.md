# 변론요지서·공판 의견서 템플릿

변론요지서는 변론 종결(결심) 단계에서 변호인의 최종 변론을 정리해 재판부에 제출하는 서면이다. 공판 진행 중 특정 쟁점에 대한 의견서도 같은 골격을 축소해 쓴다.

## 표준 골격

```
변 론 요 지 서

사    건   2026고합○○○○ 살인
피 고 인   ○○○

위 사건에 관하여 피고인의 변호인은 다음과 같이 변론의 요지를 정리하여
제출합니다.

다        음

1. 변론의 요지
2. 공소사실에 대한 피고인의 입장 (인부의 명시)
3. (부인 사건) 공소사실에 대한 다툼
   가. (쟁점 1 — 구성요건/증거능력/증명력)
   나. (쟁점 2)
4. (자백 사건 또는 예비적) 양형에 관한 의견
   가. 범행의 경위와 동기에 참작할 사정
   나. 범행 후의 정황 (피해 회복·합의·반성)
   다. 피고인의 연령·성행·환경
   라. 양형기준상의 검토
5. 결론

2026. 6. 10.

피고인의 변호인 변호사 ○○○ (인)

○○지방법원 제○형사부 귀중
```

## 노선별 골격

### 부인 사건 (무죄 변론)
3항이 본체다. 쟁점 배열의 정석:

1. **증거능력 다툼** — 위법수집증거(형사소송법 제308조의2), 자백배제(제309조), 전문법칙(제310조의2 이하, 증거 부동의 전제). 증거능력이 부정되면 증명력 판단 자체가 불필요하므로 앞세운다.
2. **증명력 다툼** — 남는 증거만으로 공소사실이 합리적 의심 없이 증명되는지(제307조 제2항). 진술 증거의 신빙성 탄핵(진술의 변천·모순·이해관계), 자백 보강법칙(제310조).
3. **법리 다툼** — 증거상 인정되는 사실을 전제로도 구성요건에 해당하지 않음(고의·인과관계 등의 법리).

각 쟁점은 3단 논증(검사 주장·증거의 요지 → 관련 법리, lbox 원문 인용 → 이 사건의 경우). 결론은 "피고인에게 무죄를 선고하여 주시기 바랍니다."

### 자백 사건 (정상 변론)
4항이 본체다. 형법 제51조의 분류를 따라 정리하되, 나열이 아니라 **사실과 자료로 뒷받침되는 항목만** 쓴다.

- 피해 회복·합의: 합의서·공탁서 등 실제 확보 자료만 인용. 처벌불원 의사가 있으면 명시(반의사불벌죄면 공소기각 주장으로 격상).
- 양형기준: SKILL.md 4.6의 2단계 확인 경로(로컬 파일 → 양형위원회 웹사이트)로 해당 범죄군의 형량 범위·특별양형인자를 확인해 "양형기준상 감경 영역에 해당" 등의 주장을 구성. 확인 불가 시 수치를 적지 않는다.
- 집행유예를 구하는 경우 형법상 요건(선고형 3년 이하 등)을 충족하는 구성인지 확인하고, 집행유예 참작 사유를 별도 단락으로.

결론은 "피고인에게 관대한 처분을 내려 주시기 바랍니다" 또는 구체적으로 "벌금형(집행유예)의 선처를 구합니다."

### 일부 인부 사건
쟁점별로 분리한다 — "제1 공소사실(사기)에 대하여는 무죄를 다투고, 제2 공소사실(횡령)에 대하여는 정상을 변론합니다"를 변론의 요지에 명시한 후, 3항과 4항을 공소사실별로 나눠 배치한다. 축소사실 인정(중한 죄명 부인)이면 "상해의 점은 인정하나 살인의 고의는 없었다"처럼 인정 범위를 정확히 긋는다.

## 작성 지침

- **2항의 인부 명시는 생략 불가**: 재판부가 서면의 성격을 즉시 파악하게 한다.
- **예비적 정상 변론의 표지**: 무죄를 다투며 정상도 변론할 때는 "가사 공소사실이 유죄로 인정된다 하더라도"의 가정적 표지를 4항 첫머리에 둔다.
- **공소사실 인용은 공소장 기재 그대로**: 요약·변형하지 않는다.
- **증거 인용**: "(증거순번 ○, 증거기록 ○면)" — 1권 초과 시 "(증거순번 ○, 증거기록 ○권 ○면)". 증인 신문은 "(증인 ○○○에 대한 증인신문조서, 공판기록 ○면)".

## 점검 포인트
- 인부(변론 노선)가 명시되어 있고 서면 전체 구조와 정합한가.
- 증거능력 주장과 증거 동의 여부가 모순되지 않는가.
- 양형 사유가 자료로 뒷받침되는가(추정 기재 0건).
- 호칭이 "피고인"으로 통일되어 있는가.
