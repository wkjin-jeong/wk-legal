# 행정쟁송 형식요건 체크리스트

신규 작성 전(선결 게이트)과 초안 완성 후(자체 점검)에 두 번 사용한다. 조문은 모두 국가법령정보 API 실측 기준이며, 의문이 생기면 ko-law-api로 재확인한다.

## A. 선결 게이트 — 작성 전 점검 (하나라도 걸리면 사용자에게 먼저 알린다)

### A-1. 처분성
- [ ] 다투는 대상이 행정소송법 제2조 제1항 제1호의 "처분등"(처분 + 행정심판 재결)에 해당하는가.
- [ ] 단순한 안내·통보·민원 회신·내부 행위가 아닌가(처분성이 의심스러우면 판례를 lbox로 확인하고, 부정 위험을 사용자에게 알린다).

### A-2. 당사자
- [ ] 피고(피청구인)가 **처분을 한 행정청**인가(행정소송법 제13조). 위임·내부위임 사안에서 처분 명의자 확인.
- [ ] 원고(청구인)에게 법률상 이익이 있는가(행정소송법 제12조, 행정심판법 제13조). 제3자 원고이면 적격 논증 준비.
- [ ] 당사자소송이면 피고가 권리주체(국가·공공단체)로 되어 있는가.

### A-3. 기간 (기산점 증빙을 자료로 확인 — 추측 금지)

| 절차 | 기간 | 근거 |
|---|---|---|
| 취소소송 | 처분이 있음을 안 날부터 90일 / 있은 날부터 1년 (불변기간) | 행정소송법 제20조 |
| 재결 경유 취소소송 | 재결서 정본 송달일부터 90일 / 재결이 있은 날부터 1년 | 행정소송법 제20조 |
| 취소심판 | 알게 된 날부터 90일 / 있었던 날부터 180일 | 행정심판법 제27조 |
| 이의신청 | 처분을 받은 날부터 30일 | 행정기본법 제36조 제1항 |
| 이의신청 후 쟁송 | 결과 통지일(또는 통지기간 만료일 다음 날)부터 90일 | 행정기본법 제36조 제4항 |

- [ ] 기산점(송달일)이 자료로 확인되는가.
- [ ] 무효등확인심판·부작위 의무이행심판은 기간 제한 없음(행정심판법 제27조 제7항). 무효확인소송은 제소기간 제한 없음 — 기간 도과 사건의 대안 노선으로 검토.
- [ ] 고지 하자가 있으면 행정심판법 제27조 제5항(오고지)·제6항(불고지) 적용 검토.

### A-4. 전심절차
- [ ] 행정심판 임의주의가 원칙(행정소송법 제18조 제1항 본문)이나, 개별 법률이 필요적 전치를 정하고 있는가(국세기본법상 심사·심판청구, 공무원 징계의 소청 등 — 해당 법률을 ko-law-api로 확인).
- [ ] 필요적 전치 사건이면 전치 충족(재결 존재) 또는 예외 사유.

### A-5. 관할
- [ ] 행정소송: 피고 소재지 관할 행정법원(행정소송법 제9조). 행정법원 미설치 지역은 지방법원 본원 합의부.
- [ ] 행정심판: 처분청에 따른 소관 행정심판위원회(중앙행정심판위원회/시·도 행정심판위원회 등).

## B. 서면별 필수 기재사항

### B-1. 소장
- [ ] 당사자·법정대리인(소송대리인) 표시
- [ ] 청구취지 — 처분의 특정(처분청·처분일자·처분 내용, 필요시 문서번호)
- [ ] 청구원인 — 처분 경위(절차 경과 포함), 위법사유, 결론
- [ ] 입증방법(처분서·송달 증빙·사전통지서·재결서 등 절차 서증 포함)
- [ ] 작성일, 법원 표시
- [ ] 가집행 선고를 구하지 않았는가(항고소송 — 성질상 부적합)

### B-2. 심판청구서 (행정심판법 제28조)
- [ ] 청구인 이름·주소, 피청구인과 위원회
- [ ] 처분의 내용, **처분이 있음을 알게 된 날**
- [ ] 청구 취지와 이유
- [ ] 행정심판 고지 유무와 그 내용

### B-3. 집행정지 신청서
- [ ] 본안 계속 사실(사건번호) — 소송의 경우
- [ ] 정지 대상의 선택(효력/집행/절차속행)과 보충성 단서 검토
- [ ] 손해 요건의 법문이 절차에 맞는가(소송 "회복하기 어려운 손해" / 심판 "중대한 손해")
- [ ] 소명 자료(소갑호증) 대응

### B-4. 의견제출서·이의신청서
- [ ] 대상 처분(예정 처분)의 표시가 통지서와 일치
- [ ] 제출 기한 확인
- [ ] 이의신청이면 행정기본법 제36조 제8항의 적용 제외 사항 여부

## C. 완성 후 공통 점검
- [ ] 인용 법령이 처분 당시 시행본인가(ko-law-api get-asof 확인 — 개정이 있으면 구법 표기).
- [ ] 판례 인용이 모두 lbox에서 확인된 것인가(추정 인용 0건).
- [ ] 호칭 통일 — 소송은 원고/피고, 심판은 청구인/피청구인, 집행정지는 신청인/피신청인.
- [ ] 본문 인용 증거번호와 입증방법(소명방법) 목록의 일치.
- [ ] 종결 문구 — 소송 "라는 판결을 구합니다" / 심판 "라는 재결을 구합니다" / 집행정지 "라는 결정을 구합니다".
- [ ] 기간 관련 기재(안 날·송달일)가 증거와 함께 본문에 들어 있는가.
