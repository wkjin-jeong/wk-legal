# 행정쟁송 문체·인용 가이드

공통 기본 문체 규칙은 플러그인 루트의 `shared/기본-문체-규칙.md`(이 skill 디렉토리 기준 `../../shared/기본-문체-규칙.md`) 참조 — 격식체("–습니다/–합니다"), 사실은 단정형, 한자 병기 금지, 항·호·목 "1. → 가. → (1) → (가)", 날짜 "2026. 6. 10." 형식, 감정적 표현·추측 표현·이모지 금지. 판례 검색·인용 정책은 `shared/판례-인용-정책.md` 참조. 여기서는 행정쟁송 특유의 규칙만 정리한다.

## 1. 호칭 — 절차별로 엄격히 구분

| 절차 | 호칭 |
|---|---|
| 행정소송 | 원고 / 피고 |
| 행정심판 | 청구인 / 피청구인 |
| 집행정지(소송) | 신청인 / 피신청인 |
| 의견제출·이의신청 | 제출인·신청인 / (수신) 행정청 |

- 한 서면 안에서 절대 혼용하지 않는다. 행정청을 지칭할 때는 "피고", "피청구인" 또는 "피고 행정청"으로 하고, "행정청"이라는 일반명사 단독 지칭은 법리 일반론 단락에서만 쓴다.
- 처분은 첫 등장 시 "피고가 2026. ○. ○. 원고에 대하여 한 ○○처분(이하 '이 사건 처분')"으로 정의하고 이후 "이 사건 처분"으로 통일한다.

## 2. 처분·재결·법령의 인용 표기

### 처분서·행정 문서
```
2026. 3. 2.자 ○○구청 ○○과-1234호 「영업정지 처분 통지」
```
- 일자 + 발신 기관·문서번호 + 문서 제목(「」). 본문에서 처분서 문구를 그대로 옮길 때는 따옴표로 감싼다.

### 재결
```
중앙행정심판위원회 2026. 4. 1.자 2026-○○○○ 재결
```

### 법령 — 처분시법 표기
- 현행본과 처분시본이 같으면 통상 표기: 행정절차법 제21조 제1항.
- 처분 후 개정이 있었으면 **구법 표기**: 구 ○○법(2026. 1. 1. 법률 제○○호로 개정되기 전의 것) 제○조. 괄호의 공포 정보는 ko-law-api get-asof의 stderr 헤더가 출력하는 직후 개정 정보를 그대로 쓴다.
- 시행령·시행규칙의 처분기준 별표: ○○법 시행규칙 [별표 ○] "행정처분 기준".
- 고시·훈령 등 행정규칙: 명칭 + 발령번호 + 발령일자(예: ○○고시 제2026-○호, 2026. 1. 1.).

### 판례
검색·인용 정책은 `shared/판례-인용-정책.md` 참조(사전 일괄 검색·판례 패키지, 원문 직접 인용, 추정 인용 금지, 검색 실패 폴백, 표기 형식). 표기 예: "대법원 2024. 5. 8. 선고 2023두12345 판결", 전원합의체 명시, 결정은 "자 ... 결정". **행정사건 사건부호(두·구·아 등)를 임의로 만들지 않는다** — lbox에서 확인한 사건번호 그대로 옮긴다. 대표 판례의 법리 설시는 따옴표로 원문 인용(길이 제한 없음), 후속 판례는 사건번호 나열("... 등 참조").

### 증거
- 소송·심판 본안: "(갑 제1호증 처분서)" — 민사와 동일.
- 집행정지 등 신청 사건: "(소갑 제1호증 ○○)".
- 행정심판 기록·처분 관련 문서의 특정 면을 인용할 때: "(갑 제3호증, 5면)".

## 3. 행정 서면 특유의 어조

- **위법성 주장은 법리 구조로, 수사는 절제**: "부당하기 짝이 없는 처분" 같은 감정 표현 대신 요건·법리·사실의 구조로 비판한다.
- **처분 경위 단락은 건조하게**: 시간순 사실만. 평가는 위법성 단락에서.
- **재량 일탈·남용 주장에서는 의뢰인의 사정을 구체적 수치로**: "막대한 손해" 대신 "월평균 매출 ○○원의 영업장을 3월간 폐쇄하게 되어 약 ○○원의 손실"처럼 소명 가능한 형태로.
- **의견제출서·이의신청서는 설득의 어조**: 쟁송 서면의 공격적 구조 대신 "…를 고려하여 주시기 바랍니다" 형의 요청 어조를 허용한다(다만 사실·법령 지적은 분명하게).

## 4. 마지막 검수 — 5단계

1. 호칭이 절차 유형에 맞게 통일되었는지(원고/피고 vs 청구인/피청구인 vs 신청인/피신청인).
2. "이 사건 처분" 정의와 일관 사용.
3. 인용 법령의 시점(처분시법)과 구법 표기의 정확성.
4. 종결 문구(판결/재결/결정)의 정합.
5. 증거 인용과 입증(소명)방법 목록의 일치.
