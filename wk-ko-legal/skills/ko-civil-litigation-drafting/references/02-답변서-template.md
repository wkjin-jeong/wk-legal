# 답변서(答辯書) 정형 템플릿

피고가 소장에 대응하여 처음 제출하는 서면. 이후 사실관계 다툼이나 항변 정리는 준비서면(`03-준비서면-template.md`)에서 한다.

## 표준 골격

```
답  변  서

사건       20○○가합○○○○ ○○○ 청구의 소
원고       ○○○
피고       △△△

위 사건에 관하여 피고는 다음과 같이 답변합니다.

청구취지에 대한 답변

1. 원고의 청구를 기각한다.
2. 소송비용은 원고가 부담한다.
라는 판결을 구합니다.

청구원인에 대한 답변

1. ...
2. ...

입증방법
1. 을 제1호증 ...

2024. 5. 8.

위 피고의 소송대리인 변호사 ○○○ (인)

○○지방법원 귀중
```

## 항목별 작성 지침

### 사건 표시
- "사건 20○○가합○○○○ ○○○ 청구의 소" — 사건번호와 사건명 그대로. 대여 사건이면 "20○○가단○○○○ 대여금"처럼 부호와 사건명이 결합.
- 원고·피고는 소장 표시와 일치시킨다(일치시키지 못하면 법원의 표시 정정 사유가 된다).

### 청구취지에 대한 답변
- **전부 다투는 경우**: "원고의 청구를 기각한다." + "소송비용은 원고가 부담한다."
- **소각하를 구하는 경우**(부적법 사유 — 소송요건 흠결, 중복제소, 권리보호이익 부재 등): "원고의 청구를 각하한다." + "소송비용은 원고가 부담한다."
- **일부만 다투는 경우**: "원고의 청구 중 ○○원을 초과하는 부분을 기각한다" 식.
- 가집행 선고에 대해서는 별도로 다툴 필요 없음(기각 판결이면 자연 해소).

### 청구원인에 대한 답변

답변의 기본 4유형:
- **자백**: 인정한다는 의미. 다툴 필요 없는 사실은 인정함이 오히려 신뢰성을 높인다.
- **부인**: 사실 자체를 다툰다.
- **부지**: 알지 못한다(자기와 직접 관련 없는 사실 한정).
- **침묵**: 명시적으로 다투지 않으면 자백 간주(민사소송법 제150조 제1항). **불필요한 침묵을 두지 않도록 모든 사실 주장에 대해 입장을 밝힌다.**

### 답변 구성 방식 두 가지

**(A) 청구원인 단락별 대응** — 소장의 단락 순서대로 인정/부인/부지를 정리.
```
1. 청구원인 1항(당사자의 지위)에 대하여
   원고가 △△의 임차인이고 피고가 △△의 임대인이라는 사실은 인정합니다.

2. 청구원인 2항(기초 사실)에 대하여
   ... 부분은 인정하나, ... 부분은 부인합니다.

3. 청구원인 3항(청구권의 발생)에 대하여
   부인합니다. 그 이유는 다음과 같습니다.
   가. ...
   나. ...
```

**(B) 사실 인부 + 항변 분리** — 먼저 사실관계 인부를 정리하고, 별도 항으로 항변을 전개.
```
1. 사실관계의 인부
   가. ... 인정.
   나. ... 부인.
   다. ... 부지.

2. 피고의 항변
   가. 변제 항변
   나. 시효 완성 항변
   ...
```

답변서 분량이 짧으면 (A), 항변이 다층적이면 (B)가 깔끔하다.

### 항변 구성 (요건사실 충족 후 발생하는 권리 소멸·저지 사실)

대표적 항변 유형:
- **변제 / 일부 변제**: 변제일자·금액·방법·증거.
- **상계**: 자동채권(피고의 원고에 대한 채권)을 특정한 후 상계의 의사표시 시점·범위 적시.
- **소멸시효 완성**: 권리 발생 시점 → 시효 기간 → 만료 시점. 시효 기간(일반 10년, 상사 5년, 단기 3년·1년 등)을 정확히 적시.
- **계약의 무효·취소·해제**: 무효·취소·해제 사유와 효과.
- **동시이행**: 상대방의 반대급부 이행을 조건으로 하는 항변.
- **공탁**: 변제공탁의 일자·법원·금액·사유.
- **권리남용·신의칙 위반**: 보충적·예외적 항변. 사실관계가 충분해야 한다.

각 항변은 **요건사실 단위로 단락**을 끊고, 사실 진술과 증거 인용을 짝지어 둔다.

### 입증방법
- 피고측은 **"을 제○호증"**. 가지번호 사용 가능.
- 본문 인용 증거번호 ↔ 입증방법 목록 일치 확인.

## markdown 출력 형식

```markdown
# 답변서

## 사건
20○○가합○○○○ ○○○ 청구의 소

## 당사자
원고 ○○○
피고 △△△

## 청구취지에 대한 답변
1. 원고의 청구를 기각한다.
2. 소송비용은 원고가 부담한다.

라는 판결을 구합니다.

## 청구원인에 대한 답변

### 1. 사실관계의 인부
**(가)** 청구원인 제1항(당사자의 지위)은 모두 인정합니다.
**(나)** 청구원인 제2항(기초 사실) 중 ... 부분은 인정하나, ...

### 2. 피고의 항변

#### 가. 변제 항변
... (요건사실 단위로 정리, 증거 인용)

#### 나. 소멸시효 완성 항변
...

### 3. 결론
원고의 청구는 이유 없으므로 기각되어야 합니다.

## 입증방법
1. 을 제1호증 ...

2026. 5. 8.

위 피고의 소송대리인 변호사 ○○○

○○지방법원 귀중
```

## 작성 시 점검 포인트
- 청구원인의 모든 사실 주장에 대해 인정/부인/부지 입장이 명시되었는가(누락은 자백 간주).
- 부인하는 사실에 대해 "그 이유는 ..."으로 적극 부인 사유를 덧붙였는가.
- 항변과 부인을 혼동하지 않았는가(부인은 요건사실 자체를 다투는 것, 항변은 요건사실 외의 별개 사실로 권리를 저지·소멸).
- 시효 항변이라면 기산점 → 시효기간 → 만료시점이 모두 명확한가. 시효 중단·정지 사유가 있는지 미리 점검.
- 상계 항변이라면 자동채권의 특정과 상계 의사표시 시점이 명확한가.
- 인용한 을호증이 모두 입증방법 목록에 있는가.
