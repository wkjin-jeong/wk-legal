# 형식요건 체크리스트

서면 작성·검토 마지막 단계에서 처음부터 끝까지 따라가며 점검한다. 한 번 봐서 잘 안 잡히는 항목이 많으니, 의심나면 두 번 본다.

## 공통 — 모든 서면에 적용

### 표제와 사건 표시
- [ ] 서면 종류(소장·답변서·준비서면)를 정확히 표제로 적었는가.
- [ ] 답변서·준비서면이라면 사건번호("20○○가합○○○○" 또는 "20○○가단○○○○")를 정확히 표시했는가.
- [ ] 사건명("○○○ 청구의 소")이 청구의 법적 성격에 부합하는가.
- [ ] 준비서면이 두 번째 이상이라면 차수("(2)", "(3)" 등) 또는 변호사 통상 관행에 따른 차수 표시.

### 당사자 표시
- [ ] 자연인의 성명·주소가 정확한가(주민등록상). 송달장소가 다르면 별도 명시.
- [ ] 법인의 명칭이 등기부 그대로인가("주식회사" 위치, "(주)" 약칭 사용 여부 일관성).
- [ ] 법인 대표자 표시(대표이사 ○○○).
- [ ] 다수 당사자(원고1·원고2 / 피고1·피고2)의 호칭이 본문에서 일관되게 유지되는가.
- [ ] 미성년자·피한정후견인 등의 법정대리인 표시.
- [ ] 소송대리인(변호사)을 선임했다면 변호사 표시.

### 본문 일관성
- [ ] 호칭 — "원고", "피고"로 통일(피고인·청구인·신청인 등 다른 절차 용어 혼입 여부).
- [ ] 문체 — 격식체("–습니다/–입니다/–합니다"체) 통일. 구어체·반말 혼입 금지.
- [ ] 인명·지명·법인명이 처음 등장한 표기를 일관되게 유지.
- [ ] 날짜 표기 — 연·월·일을 모두 적을 때는 "2024. 5. 8." 형식으로 통일. 연·월·일 중 하나만 쓰는 경우(예: 연도만 / 월·일만)에는 "2024년", "5월 8일"식으로 한글 단위(년/월/일)를 붙인다. 한 서면 안에서 일관되게 유지.
- [ ] 금액 표기 — 한글·아라비아숫자 병기 또는 일관된 한 방식("금 1억 원(1억 원)" / "100,000,000원").

### 증거 인용
- [ ] 본문에서 인용한 모든 증거가 입증방법 목록에 있는가.
- [ ] 입증방법 목록의 모든 증거가 본문에서 인용되는가(목록만 있고 본문 무인용은 분량 낭비).
- [ ] 증거번호 형식 — 원고 "갑 제○호증", 피고 "을 제○호증". 가지번호("의 1, 의 2") 사용 시 일관.
- [ ] 호증명이 증거의 성격을 정확히 표현하는가.

### 작성일과 법원
- [ ] 작성일이 사건 진행 시점에 부합하는가(미래 일자 또는 현저한 과거 일자가 아닌지).
- [ ] 변호사 또는 본인 서명란.
- [ ] 법원 표시 — 답변서·준비서면이라면 재판부까지("○○지방법원 제○부 귀중").

> **부속서류 점검은 본 체크리스트 범위 밖이다.** 본 skill의 출력에는 부속서류 항목이 포함되지 않으며, 인지·송달료·부본 통수·법인등기부등본 등 사무적 첨부서류는 변호사가 별도로 관리한다.

## 소장 전용 점검

### 청구취지의 특정성
- [ ] **금전 청구**: 원금 + 기산일 + 종기("다 갚는 날까지") + 이율 + "비율로 계산한 돈".
  - [ ] 변제기까지의 약정이자와 변제기 이후의 지연손해금을 구분해야 한다면 단계별 표시.
  - [ ] 이율이 약정이율인지 법정이율(연 5% 민사 / 연 6% 상사)인지 또는 소송촉진법 이율(연 12%)인지 명시.
- [ ] **부동산 인도/명도**: 부동산을 등기부 그대로 특정(소재지·지번·지목·면적, 건물은 동·호수·구조까지).
- [ ] **등기 청구**: 등기 표시(부동산 + 등기소 + 접수일 + 접수번호 + 등기 종류 + "이전등기/말소등기절차를 이행하라").
- [ ] **확인의 소**: 확인 대상이 명확히 특정되어 있는가.
- [ ] **소송비용 부담**: "소송비용은 피고가 부담한다."
- [ ] **가집행 선고**: 적합한 청구인지(의사표시·등기 일부는 부적합).
- [ ] 종결문 — "라는 판결을 구합니다."

### 청구원인의 정합성
- [ ] 청구원인이 청구취지의 모든 요소(원금·이자·기산일 등)를 지탱하는가.
- [ ] 요건사실이 빠짐없이 진술되었는가(쟁점별 논증구조 reference 참고).
- [ ] 청구의 법적 근거(조문)가 결론 부분에서 분명한가.

### 관할
- [ ] 토지관할 — 피고 보통재판적, 의무이행지(지참채무·추심채무), 부동산 소재지 등.
- [ ] 사물관할 — 5억 원 이하 단독, 초과 합의(「민사 및 가사소송의 사물관할에 관한 규칙」 제2조. 5억 기준은 2022년 1월 28일 개정 — 사용자에게 사건 시점 기준 관할 확인).
- [ ] 합의관할·전속관할 — 약정 또는 법령상 전속관할 검토.

### 인지·송달료
- [ ] 인지액 산정 근거(소가) 점검.
- [ ] 송달료 — 소장 부본 송달 + 변론기일 통지 등 산정.
- [ ] 변호사가 별도 산정한 경우 본문에는 적시하지 않으나, 변경 사항 논평에서 인지·송달료 산정 검토를 권고.

## 답변서 전용 점검

- [ ] 청구취지에 대한 답변이 명확한가("기각" / "각하" / 일부 답변).
- [ ] 청구원인 모든 단락에 대해 인정/부인/부지 입장이 표시되었는가(누락은 자백 간주).
- [ ] 부인 사실에 대해 적극적 부인 사유를 덧붙였는가.
- [ ] 항변과 부인을 혼동하지 않았는가.
- [ ] 시효 항변 — 기산점·시효기간·만료시점이 모두 명확한가.
- [ ] 상계 항변 — 자동채권의 특정과 상계 의사표시 시점이 명확한가.
- [ ] 동시이행 항변 — 반대급부의 내용이 정확히 특정되어 있는가.
- [ ] 반소·소송고지 검토 — 단순 답변을 넘어 별도 소송행위가 필요한지(본문 외 별지 신청서로 처리하되 사용자에게 안내).

## 준비서면 전용 점검

- [ ] 작성자(원고/피고) 표시가 첫 줄에 분명한가.
- [ ] 직전 서면(소장·답변서·이전 준비서면)의 호증번호 다음부터 새 호증번호를 매겼는가(중복·건너뜀 없음).
- [ ] 상대방 주장의 요지를 정확히 인용했는가(왜곡·축소 금지).
- [ ] 각 반박 단락이 **3단 논증 형식**(상대방 주장의 요지 → 관련 법리 → 이 사건의 경우)을 갖추었는가.
- [ ] 법리 단락의 판례 인용을 **lbox-case-search**로 적극 보강했는가. 핵심 쟁점에 대표 판례 1건 + 같은 취지 후속 판례 1~2건이 인용되어 있는가.
- [ ] 핵심 쟁점에 분량을 집중하고, 부수 주장은 짧게 처리했는가.
- [ ] 청구의 변경(교환적·추가적)에 해당하는 사항이 있다면 별도 청구취지 변경 신청서를 안내했는가.
- [ ] 결론 단락에서 이 서면이 향하는 결과를 명시했는가.

## 자주 빠뜨리는 사항 — 마지막 한 번 더 점검

- [ ] **법인등기부등본의 첨부 의도성** — 법인 당사자가 있을 때마다 첨부해야 한다.
- [ ] **인용한 판례·법령 번호의 정확성** — 사건번호·선고일자·판결 종류(전합·일반)가 정확한가. **본 skill에서 인용한 모든 판례는 lbox-case-search로 검증·확보된 것이어야 한다.** 추정·기억에 의한 사건번호 기재가 발견되면 lbox 재검색으로 교체하거나 삭제한다.
- [ ] **별지 사용 권장 항목** — 부동산 표시, 손해 산정 내역, 이자 계산표, 다수 당사자 명단 등은 별지로 분리하면 가독성 ↑.
- [ ] **서면 분량의 적정성** — 핵심 분량 대비 부수 주장이 과도하지 않은지. 분량은 미덕이 아니다.
- [ ] **숫자·금액의 단순 오타** — 본문, 청구취지, 입증방법 사이 금액 일치.
- [ ] **이전 서면과의 정합성** — 답변서·준비서면이라면 직전 서면에서 인정한 사실을 뒤늦게 부인하고 있지 않은가(자백 철회는 까다롭다).
